## 🔬 WooJJam의 스프링 연구실

해당 저장소는 스프링을 사용한 삭습 기록과 자료를 정리해나가는 공간입니다.
각 모듈 별로 특정 주제나 기술에 대한 공부 및 연구 내용을 포함합니다.

## 🧬 Module

기술별 모듈들은 Multi Module을 기반으로 작성됩니다.
- **aop**: Spring의 AOP를 학습
- **concurrency**: Spring에서 다양한 동시성 제어 방법을 연구하는 모듈
- **infrastructure**: Spring과 인프라 설계를 통합하여 배포 테스트 모듈
- **jacoco-test-reports**: JaCoCo 로 테스트 커버리지를 측정하기 위한 모듈
- **labs-domain**: JPA를 사용한 공통 모듈로 모든 모듈에서 사용가능한 영역
- **monitoring** 각종 모니터링 툴을 학습하는 모듈
- **notification**: 다양한 알림 시스템을 연구하는 모듈
- **pratical-testing**: Junit5를 이용한 테스트 코드 작성을 학습하는 모듈
- **schedule**: Spring에서 제공하는 스케줄링에 대해서 학습하는 모듈

새로운 모듈 추가시 해당 모듈의 설명을 수정할 예정입니다.
